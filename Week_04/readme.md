# 🎓 Student Management System

A Flask-based Student Management System developed using Python, SQLite, Bootstrap, HTML, CSS, JavaScript, and Jinja2. This project helps manage student records through a simple and user-friendly web interface.

## 🚀 Features

- Dashboard with student statistics
- Add new students
- View all students
- Update student details
- Delete student records
- Search student by Roll Number
- Search students by Department
- Department Distribution Chart (Chart.js)
- Top 5 Performers
- CSV Export
- Responsive UI using Bootstrap

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Jinja2
- Chart.js

## ▶️ How to Run

1. Clone the repository
2. Install the required packages

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python main.py
```

4. Open your browser and visit

```
http://127.0.0.1:5000
```
