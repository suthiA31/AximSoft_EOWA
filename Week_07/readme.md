# 🏨 Hotel Booking Analytics Dashboard

An interactive Hotel Booking Analytics Dashboard developed using **Python, Flask, Pandas, Plotly, HTML, CSS, Bootstrap, and Jinja2**. The application provides data visualization, booking insights, customer analytics, revenue analysis, statistical reports, and dataset exploration through an interactive web interface.

---

## 📌 Project Overview

This project analyzes hotel booking data and presents meaningful business insights through interactive dashboards and visualizations.

The dashboard helps users:

- Analyze booking trends
- Monitor revenue performance
- Understand customer behavior
- Explore datasets interactively
- Perform statistical analysis
- Download processed datasets
- Access project notebooks
